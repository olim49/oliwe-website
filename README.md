# Oliwe — Site vitrine

Site vitrine développé en **HTML / CSS / JavaScript** pour présenter les services de création de sites web et supports visuels proposés par Oliwe.

## 🚀 Fonctionnalités
- Page d’accueil moderne
- Section Services
- Section Pour qui
- Portfolio dynamique (JS)
- Formulaire de contact
- Mode Dark/Light
- Animations AOS
- Design responsive

## 🛠️ Technologies utilisées
- HTML5
- CSS3
- JavaScript
- AOS (animations)
- GitHub Pages (déploiement)
